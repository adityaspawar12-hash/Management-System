# Management-System

Lottery Management System is a secure web-based platform for automating lottery cycles. Built with Python and Flask, it features a dual-interface for "Players" and "Admins". It handles user registration, ticket generation, and fair winner selection through randomized logic, ensuring transparency and data integrity in every draw.

## Features

User Management: Register and manage participant profiles.
Ticket Generation: Automated generation of unique lottery tickets.
Fair Draw Mechanism: Built-in randomization logic for unbiased winner selection.
Dashboard:  View active draws, total participants, and historical winners.
Transaction Logs:  Track ticket purchases and prize distributions.

## Tech Stack

Language: Python
Framework: Flask
